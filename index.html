使い方ガイド

📝 データの入力方法
  1. ホーム画面の 「今日の入力」 をタップ
  2. 日付が自動でセットされます（変更も可能）
  3. 生産数・B.C反・胴継反 を入力
  4. B.C反%・胴継反%はリアルタイムで自動計算されます
  5. 「保存」 をタップして完了
📅 過去データの修正
  1. 入力画面で 「‹」「›」 をタップして日付を変更
  2. または日付部分をタップしてカレンダーから選択
  3. 既存データがある日は自動で読み込まれます
  4. 修正して「保存」すると上書き確認が表示されます
📊 グラフの見方
  • 青：生産数
  • 黄：B.C反
  • 橙：胴継反
  • データなし（土日・祝日等）はバーが表示されません
🖨️ 印刷・PDF保存
  1. 「月次グラフ」をタップ
  2. 右上の 🖨️ で印刷ダイアログが開きます
  3. PDFで保存するには印刷先で 「PDFに保存」 を選択
  4. 印刷は横向き（A4ランドスケープ）で最適化済みです
⚠️ 注意事項
  • 日曜日はデータ入力できません
  • 週集計は月〜土の合計が自動計算されます
  • データはFirebaseに保存されます（全スタッフ共有）
